# Thư mục practice

Dùng cho **file dùng chung** giữa các bài (ví dụ model, interface). Phần **Thực hành** chi tiết (File + Cách làm) nằm theo **từng bài học** trong `src/app/lessons/`.

| File trong practice/ | Dùng trong bài |
|----------------------|----------------|
| `practice/models/product.model.ts` | **Bài 1** — Interface `Product` (TypeScript). |

**Làm thực hành ở đâu:** Mở `src/app/lessons/01-component-basic/README.md` … `08-testing-ssr/README.md` tương ứng với từng bài. Mỗi README có đầy đủ **File(s)** và **Cách làm** từng bước. Trong lộ trình `LO_TRINH_HOC_ANGULAR.md`, mỗi mục Thực hành chỉ ghi “→ Làm theo **Bài N**” và đường dẫn README.
